# Business Sales Analysis Dashboard üìä

An interactive dashboard that explores sales data to answer important business questions.

### What This Dashboard Shows
- Which season has the highest sales
- How promotions affect sales volume
- Which product positions make the most money
- Top selling products by revenue

### Features
- click on the home to read about the data.
- Filter data using the sidebar (choose any category like Season, Promotion,terms, etc.)
- See total revenue, sales volume, and number of records
- Beautiful charts made with Plotly
- Easy to read and understand insights

### Tools Used
- Python
- Streamlit (for the dashboard)
- Pandas (for data handling)
- Plotly (for charts)
